# Consolidation Rules – Linux Desktop Chat

**Effective:** 2026-03-16  
**Purpose:** Prevent regression to hybrid/duplicate state. Enforce single source of truth.

---

## 1. Documentation

- **Docs root:** No new `.md` files in `docs/` root except (enforced post-migration):
  - `README.md`
  - `00_map_of_the_system.md`
  - `SYSTEM_MAP.md`
  - `TRACE_MAP.md`
  - `FEATURE_REGISTRY.md`
- **New project docs:** Place in numbered sections `docs/0x_*/` per taxonomy.
- **User help:** No new user-facing help content in `docs/`. All help content goes in `help/`.

## 2. Assets

- **Icons:** `assets/icons/` is canonical. No duplicated icons in `app/resources/icons/` or `app/gui/icons/svg/`.
- **Themes:** `assets/themes/` is canonical.
- **Media:** `assets/media/` is canonical.

## 3. Source Code

- **Runtime source:** `app/` is canonical. `src/` is a wrapper only until full migration.
- **Legacy UI:** No new code in `app/main.py`. Legacy GUI lives in `archive/`.
- **Entry point:** `main.py` → `run_gui_shell.py` is the standard path.

## 4. Help System

- **Help content:** `help/` is canonical. `docs/` is fallback only for backward compatibility.
- **HelpIndex:** Loads `help/` first, then `docs/` for unmigrated topics.

## 5. Generated Artifacts

- **System map:** `docs/SYSTEM_MAP.md` — generated by `tools/generate_system_map.py`.
- **Trace map:** `docs/TRACE_MAP.md` — generated by `tools/generate_trace_map.py`.
- **Cache:** `__pycache__/`, `.pytest_cache/` — not tracked. Add to `.gitignore` if missing.
- **Zip artifacts:** No `*.zip` in `docs/` (e.g. `docs/qa.zip` removed).

## 6. Archive

- **Legacy code:** Deprecated entry points and GUI in `archive/`.
- **Deprecated docs:** Move to `archive/deprecated_docs/` when superseded.

---

## Map Documents (Core Orientation)

| Document | Role | Generated? |
|----------|------|------------|
| `00_map_of_the_system.md` | Human-readable orientation | No |
| `SYSTEM_MAP.md` | Structural map (app, workspaces, services) | Yes |
| `TRACE_MAP.md` | Code ↔ Help ↔ Tests traceability | Yes |
