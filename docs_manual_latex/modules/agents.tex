\section{Agents}

\subsection{1. Fachsicht}

Das \textbf{Agenten-Subsystem} verwaltet \textbf{Profile}, \textbf{Aufgaben} und \textbf{Orchestrierung} (Planner, Delegation, Execution). Nutzer w\textbackslash\{\}"ahlen Agenten im Chat oder nutzen \textbf{Operations \$\textbackslash\{\}rightarrow\$ Agent Tasks} bzw. \textbf{Control Center \$\textbackslash\{\}rightarrow\$ Agents}. Slash-Command \textbf{`/delegate`} aktiviert in der Command-Schicht \texttt{use\_delegation=True} (\texttt{app/core/commands/chat\_commands.py}); die weitere Ausf\textbackslash\{\}"uhrung liegt in den Agenten- und Chat-Pfaden.

\textbf{Paket:} \texttt{app/agents/}

\subsection{3. Prozesssicht}

\subsubsection{3.1 Chat mit Agentenprofil}

\begin{verbatim}
User w\"ahlt Agent (GUI)
        ?
        v
AgentProfile (system_prompt, assigned_model / Rolle)
        ?
        v
Gleicher Chat-Pfad wie Modul Chat $\rightarrow$ Provider
\end{verbatim}

\subsubsection{3.2 Delegation (Einstieg Slash-Command)}

\begin{verbatim}
Eingabe: /delegate <Aufgabe>
        ?
        v
parse_slash_command $\rightarrow$ use_delegation=True, remaining_text=Aufgabe
        ?
        v
ChatService / Agenten-Pipeline
  (Planner $\rightarrow$ Task Graph $\rightarrow$ Delegation Engine $\rightarrow$ Execution Engine)
  --- konkrete Aufrufkette in app/agents/*.py
\end{verbatim}

\textbf{Relevante Dateien (Auswahl):} \texttt{planner.py}, \texttt{delegation\_engine.py}, \texttt{execution\_engine.py}, \texttt{orchestration\_service.py}, \texttt{agent\_task\_runner.py}.
