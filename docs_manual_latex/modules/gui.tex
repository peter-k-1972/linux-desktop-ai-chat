\section{GUI}

\subsection{1. Fachsicht}

Die \textbf{GUI} ist die \textbf{PySide6-Shell} der Anwendung: Hauptfenster, Bereichsumschaltung, Workspaces je Screen, \textbf{Navigation} (Sidebar, Breadcrumbs, Command Palette), \textbf{Inspector} (rechts), \textbf{Monitors} (unten) und dom\textbackslash\{\}"anenspezifische \textbf{Panels}. Zentraler Code unter \textbf{`app/gui/`} (nicht \texttt{app/ui/} --- dieses Verzeichnis existiert im aktuellen Tree nicht, vgl. \texttt{docs/DOC\_GAP\_ANALYSIS.md}).

\subsubsection{4.3 Panels}

\textbf{Panels} sind komponentenweise UI-Teile innerhalb von Workspaces (z. B. unter \texttt{app/gui/domains/operations/chat/panels/} --- \texttt{ChatConversationPanel}, \texttt{ChatInputPanel}, \texttt{ChatContextBar}, \texttt{chat\_navigation\_panel}, \textbackslash\{\}ldots\{\}). Es gibt keine einzelne zentrale ?Panel-Registry\texttt{}-Datei; die Einbindung erfolgt \textbackslash\{\}"uber die jeweilige \texttt{*Workspace}-Klasse.

\subsubsection{4.4 Inspector \& Backends}

\begin{itemize}
\item \textbf{Inspector:} \texttt{app/gui/inspector/} --- z. B. \texttt{ChatContextInspector} f\textbackslash\{\}"ur Chat-Kontext.
\item \textbf{ChatBackend / KnowledgeBackend:} \texttt{app/gui/chat\_backend.py}, \texttt{app/gui/knowledge\_backend.py} --- Anbindung an Services.
\end{itemize}
