# Hilfe: Chat-Kontext

Dieses Kapitel beschreibt, wie **Kontext** und **Modelle** zusammenspielen.

## Überblick

- **Kontext**: zusammengefasste Historie und RAG-Treffer
- **Modell**: das gewählte Sprachmodell
- **Agent**: optionale Spezialrolle

## Häufige Fragen

1. Wo ändere ich das Modell?
2. Wie groß darf der Kontext sein?

> Tipp: Unter *Einstellungen → Modelle* finden Sie die Verfügbarkeit.
