# Beispiel-Dokument für RAG-Tests

Dieses Dokument enthält Beispielinhalt für die RAG-Pipeline-Tests.

## Abschnitt 1: Einführung

Das Linux Desktop Chat Projekt bietet eine integrierte Chat-Oberfläche mit Agenten,
RAG-Unterstützung und Debug-Funktionen.

## Abschnitt 2: Technische Details

- PySide6 für die UI
- Ollama als LLM-Backend
- ChromaDB für Vektor-Speicherung
- SQLite für Persistenz

## Abschnitt 3: Agenten

Agenten können nach Abteilungen organisiert werden: Planning, Research, Development,
Media, Automation und System.
